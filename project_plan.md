# The Compound - Personal Blog

## 1. Project Description
A personal blog by "The Compound Kid," a 15-year-old Nigerian who builds apps, goes to church, fights private battles, and thinks too much. The blog serves Nigerian youth aged 13–25.

## 2. Page Structure
- / - Home (hero, latest posts, about preview, footer)
- /addiction-files - Addiction Files blog category page
- /game - Compound Life game page (HTML5 Canvas)
- /community - The Compound Floor community feed
- /profile - Profile page (social media style)
- /messages - Private messaging page
- /source - Open Source code browser
- /admin/* - Admin system pages

## 3. Core Features
- [x] Homepage with navigation, hero, posts grid
- [x] Addiction Files category page
- [x] Game page with Canvas engine
- [x] Community page with post composer and feed
- [x] Profile page with tabs
- [x] Messages with real-time chat
- [x] Admin system (dashboard, posts, settings)
- [x] Open Source page with file browser and code converter

## 4. Tech Stack
- React 19 + TypeScript + TailwindCSS 3
- Supabase (Auth, Database, Edge Functions)
- Vite 8
